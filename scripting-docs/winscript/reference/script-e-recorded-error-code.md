# <a name="scripterecorded-error-code"></a>SCRIPT_E_RECORDED 错误代码
脚本引擎和主机之间传递了错误。 主机需要传递给调用方的错误代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
