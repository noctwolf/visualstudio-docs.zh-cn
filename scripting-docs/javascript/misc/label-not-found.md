# <a name="label-not-found"></a>未找到标签
您所做的引用不存在的标签。 标签可用于将标记块的代码，但在指定作用域内，必须是唯一的。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 检查以确保标签名称拼写正确。  
  
- 请确保所有标签引用都所引用的标签都已经在当前作用域 （这包括向前的定义） 定义。
