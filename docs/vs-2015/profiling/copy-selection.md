# <a name="copy-selection"></a>复制选择
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

若要从“报告”选项卡复制整个调用堆栈，请单击“复制”  。 然后，可以在任何支持该操作的程序中粘贴该调用堆栈。
