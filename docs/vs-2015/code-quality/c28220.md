# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28220： 批注中的参数的批注预期整数表达式  
  
 此警告意味着批注参数应为整数表达式，并且遇到了一些其他类型。 这通常表示存在错误编码的批注宏。
