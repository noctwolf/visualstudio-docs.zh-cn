# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28230:参数类型没有成员。  
  
 此警告意味着为批注的参数尝试访问 `struct`、`class` 或 `union`，并且所命名的成员不存在。
