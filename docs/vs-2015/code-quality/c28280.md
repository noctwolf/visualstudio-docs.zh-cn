[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28280](https://docs.microsoft.com/visualstudio/code-quality/c28280)。  
  
警告 C28280： 对于符号，end 发现了没有匹配的 begin  
  
 批注语言支持 begin 和 end (`{`和`}`C 中) 构造，并且此配对收到不平衡。 如果使用了宏，则此情况可避免。
