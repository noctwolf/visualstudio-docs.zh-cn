# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28212:批注所需的指针表达式  
  
 此警告意味着所编号的批注参数（不是要批注的函数）应该是指针或数组类型，但却遇到某些其他类型。 此批注需要更正。
