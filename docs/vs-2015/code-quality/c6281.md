# <a name="c6281"></a>C6281
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 6281-运算顺序不正确： 关系运算符具有更高的优先级高于位运算符  
  
 此警告表示的运算符优先级可能的错误。 这可能会产生不正确的结果。 应检查优先级，并使用括号明确指定优先级。 关系运算符 (\<，>， \<=、 > =、 = =、 ！ =) 具有更高的优先级高于位运算符 (& &#124; ^)。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if (FORMAT & TYPE != input)  
  {  
    // code...  
  }  
}  
```  
  
 下面的代码使用括号来更正此警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if ((FORMAT & TYPE) != input)  
  {  
    // code...  
  }  
}  
```
