**ID：** Microsoft.VisualStudio.Workload.TestAgent

**描述：** 支持运行自动测试和远程加载测试

### <a name="components-included-by-this-workload"></a>此工作负载所包含的组件

--- | --- | --- | ---
Microsoft.VisualStudio.ComponentGroup.TestTools.TestAgent | 测试代理核心功能 | 15.0.27019.1 | 必需
