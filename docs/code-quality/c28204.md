# <a name="c28204"></a>C28204
警告 C28204:\<函数 >： 只有一个的此重载，并在一个\<文件名 > (\<行 >) 的批注\<参数名称 >： 必须批注同时还是两者皆否。  
  
 当批注中存在错误时将报告此警告。
