# <a name="c28160"></a>C28160
警告 C28160:错误批注

 当遇到 `__drv_error` 批注时，将报告此警告。 此批注用于标记应修复的编码做法，并且可以与 `__drv_when` 批注一起用来指示特定的参数组合。
