# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
将枚举序列重置到开头。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>返回值  
 返回 `S_OK`。
