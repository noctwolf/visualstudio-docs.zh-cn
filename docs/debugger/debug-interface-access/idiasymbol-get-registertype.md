# <a name="idiasymbolgetregistertype"></a>IDiaSymbol::get_registerType
检索注册类型。

## <a name="syntax"></a>语法

```C++
HRESULT get_registerType(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out]一个指向`DWORD`保存注册类型。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
