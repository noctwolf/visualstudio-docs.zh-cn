# <a name="idiasourcefilegetfilename"></a>IDiaSourceFile::get_fileName
检索源文件名。

## <a name="syntax"></a>语法

```C++
HRESULT get_fileName ( 
   BSTR* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out]返回源文件名。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为返回错误代码。
