# <a name="idiaenumframedatagetcount"></a>IDiaEnumFrameData::get_Count
检索帧数据元素的数目。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 pRetVal  
 [out]返回帧数据元素的数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。
