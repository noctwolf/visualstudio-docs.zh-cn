# <a name="window-menu-commands"></a>“窗口”菜单命令
**窗口**菜单提供用于管理 Spy + + windows 的命令。

 **级联**重新排列的级联模式中的所有窗口。

 **水平平铺**重新排列水平平铺模式中的所有窗口。

 **垂直平铺**以重新排列所有窗口垂直平铺模式。

 **排列图标**重新排列视图图标，如果 windows 已最小化。

 **关闭 / 关闭所有**关闭 Spy + + 中的一个或所有窗口。

 **刷新**刷新当前所选的视图。

 **窗口**允许选择特定的窗口。
