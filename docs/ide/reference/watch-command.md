# <a name="watch-command"></a>“监视”命令
创建并打开指定“监视”  窗口的实例。 可使用“监视”窗口计算变量、表达式或寄存器的值，然后编辑这些值并保存结果  。

## <a name="syntax"></a>语法

```cmd
Debug.Watch[index]
```

## <a name="arguments"></a>自变量

`index`\
必需。 监视窗口的实例数。

## <a name="remarks"></a>备注

`index` 必须为整数。 有效值为 1、2、3 或 4。

## <a name="example"></a>示例

```cmd
>Debug.Watch1
```
