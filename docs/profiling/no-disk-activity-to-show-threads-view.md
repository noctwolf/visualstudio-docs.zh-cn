# <a name="no-disk-activity-to-show-threads-view"></a>没有要显示的磁盘活动（线程视图）
无法显示磁盘 I/O 通道中当前时间范围的磁盘活动。 如果缩小或滚动，可能会显示磁盘活动。 此外，请确保没有磁盘通道处于隐藏状态。
