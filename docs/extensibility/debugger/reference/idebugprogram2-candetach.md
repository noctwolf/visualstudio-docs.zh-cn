# <a name="idebugprogram2candetach"></a>IDebugProgram2::CanDetach
确定是否调试引擎 (DE) 可以从程序中分离。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT CanDetach(  
   void  
);  
```  
  
```csharp  
int CanDetach();  
```  
  
## <a name="return-value"></a>返回值  
 如果可以分离，返回`S_OK`; 否则为返回错误代码。 返回`S_FALSE`如果 DE 无法从程序中分离。
