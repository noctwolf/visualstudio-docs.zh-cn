# <a name="suspendtracking"></a>SuspendTracking
在当前上下文中暂停跟踪。

## <a name="syntax"></a>语法

```cpp
HRESULT WINAPI SuspendTracking(void);
```

## <a name="return-value"></a>返回值
 如果跟踪暂停，则返回带 SUCCEEDED 位集的 HRESULT   。

## <a name="requirements"></a>要求
 **标头：** FileTracker.h
